## Getting started

Run basic commands after cloning the repository

```
npm install
```

```
npm run dev
```

```
composer install
```

```
php artisan migrate
```

```
cp .env.example .env
```

```
php artisan key:generate
```

Create some test articles to display at main view

```
php artisan db:seed
```

**Then go to /register to create a test account to be able to comment articles**
